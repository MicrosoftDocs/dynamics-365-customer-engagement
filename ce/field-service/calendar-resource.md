# Apply a calendar to a resource

Create a work hours template to define working hours for a set of resources. Then assign it to those resources so their calendars reflect those working hours. For example, create a template for your standard working hours and apply it to all your technicians. Or create a template for a specific team or location and apply it to the resources in that team or location.

## Create a work hours template

1. In Field Service, change to the **Resources** area, and select **Resources**.

1. Select **Work Hours Templates** and then select **New**.
  
1. Enter a name and description for the template. If you have resource that you want to use as a template, select them.

1. Select **Save**.

1. Select the **Calendar** tab. Verify or change the working hours for the template. Select **Save & Close**.

## Set the calendar for a resource

1. In the **Resources** area, select **Resources**.  
  
1. Select the box next to the resource you want to set the calendar for. You can select more than one resource, but because of performance constraints, you can select up to 25 resources.  
  
1. Select **Set calendar**.  
  
   :::image type="content" source="media/resource-set-calendar.png" alt-text="Screenshot of a selected resource highlighting the Set calendar button.":::

1. Choose the work hours template you want to apply.  
  
1. Select **Apply**.
