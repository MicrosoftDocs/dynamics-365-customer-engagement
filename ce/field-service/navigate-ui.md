# Navigate the user interface

Learn how to navigate the Dynamics 365 Field Service web app.

## Navigate using the site map

The following image shows the site map, also called the navigation panel in the Field Service web app. Use site map helps you navigate to different areas of the app.

:::image type="content" source="media/navigate-site-map.png" alt-text="Screenshot in the Field Service web app highlighting key navigation tools." lightbox="media/navigate-site-map.png":::

| Number | Description |
|--------|-------------|
| 1      | **Collapse/expand**: Open or collapse the site map. When collapsed, the site map shows icons for the different pages in the app. |
| 2      | **Recent records/Pinned records**: Access records you recently worked on or records you pinned. Use the **Recent** list and select the pin next to a page to add it to **Pinned records**. |
| 3      | **Page navigation**: Open different pages in the app to do your work. |
| 4      | **Area**: Switch between different work areas in the app such as Service, Settings, or Resources. |
| 5      | **App selector**: Switch to another app. |
| 6      | **Command bar**: Access the command bar for quick actions. The available commands vary based on your role and the page you're on. |

## Move between apps

Use the app selector to switch between apps.

1. Select the current app name in the app selector.

   :::image type="content" source="media/navigate-app-selector.png" alt-text="Screenshot of the app selector.":::

1. On the **Apps** page, select the app you want to open. You see only apps that your organization installs.

   :::image type="content" source="media/navigate-switch-apps.png" alt-text="Screenshot of the Apps page.":::

## Work with list views

When you select most pages in the Field Service web app, a list of records appears. For example, when you select **Accounts** in the site map, a list of accounts appears.

:::image type="content" source="media/navigate-list-view.png" alt-text="Screenshot in the Field Service web app showing a list of accounts.":::

| Number | Description |
|--------|-------------|
| 1      | **Go back**: Return to the previous page. |
| 2      | **Select records**: Select one or more records by using the check box. Depending on the page, you can apply a single action to all selected records by using the command bar.|
| 3      | **Open a record**: Select a record to open its record and view the record details. |
| 4      | **Sort or filter the list**: Select the arrow to sort or filter the list by values in that column. |
| 5      | **View selector**: Change the current view. |
| 6      | **Edit columns**: Add, remove, or reorder columns. Learn more in [Column editor](/power-apps/user/grid-filters-advanced#select-columns-in-the-column-editor) |
| 7      | **Edit filters**: View, add, or modify filters applied to the current view. Learn more in [Filter editor](/power-apps/user/grid-filters-advanced#select-filter-conditions-in-the-filter-editor) |
| 8      | **Filter by keyword**: Enter text in the search field to show only records that match the keyword in the current view. |

## Work with record views

Record views show all available data and settings for a single record. They're also known as form views.

To open a record view, select a record from a list view. You can also open a record view by selecting a link from a related record.

:::image type="content" source="media/navigate-record-view.png" alt-text="Screenshot in the Field Service web app showing an account record.":::

| Number | Description |
|--------|-------------|
| 1      | **Tabs**: Record views are divided into tabs. Each tab contains a group of related fields from the record. The current tab appears underlined. Mention the goal before the action. To view its fields, select a tab. |
| 2      | **Related**: Most records include a **Related** tab. Use this drop-down list to view records related to the current record. |
| 3      | **Heading bar**: Some record views display key fields in the heading bar, such as the record owner. |
| 4      | **View and edit all field values**: The main body of the record view, shows fields for the selected table. Required fields are marked with a red asterisk, and you can't save the record without valid values. Fields marked with a blue plus sign are especially important or recommended, but aren't strictly required. Fields showing a lock icon are read-only and can't be edited. |
