# Navigate the user interface

Learn how to navigate the Dynamics 365 Field Service web app.

## Navigate using the site map

The following image shows the site map or navigation panel of the Field Service web app. The site map helps you navigate to different areas of the app.

:::image type="content" source="media/navigate-site-map.png" alt-text="Screenshot in the Field Service web app highlighting key navigation tools." lightbox="media/navigate-site-map.png":::

| Number | Description |
|--------|-------------|
| 1      | **Collaps/expand**: Open or collapse the site map. The collapsed map shows icons for the different pages in the app. |
| 2      | **Recent records/Pinned records**: Access the recent records you worked on or the records you pinned. Use the **Recent** list and select a pin next to a page to add it to the **Pinned records**. |
| 3      | **Page navigation**: Access different pages in the app to do your work. |
| 4      | **Area**: Use area to switch between different work areas in the app. For example, Service, Settings, or Resources. |
| 5      | **App selector**: Use the app selector to switch to another app. |
| 6      | **Command bar**: Access the command bar for quick actions. The available commands vary based on your role and the page you're on. |

## Move between apps

Use the app selector to switch between apps.

1. Select the current app name in the app selector.

   :::image type="content" source="media/navigate-app-selector.png" alt-text="Screenshot of the app selector.":::

1. On the **Apps** page, select the app you want to switch to. You see only apps that your organization installed.

   :::image type="content" source="media/navigate-switch-apps.png" alt-text="Screenshot of the Apps page.":::

## Work with list views

When you select most pages in the Field Service web app, a list of records appears. For example, if you select **Accounts** in the site map, a list of accounts appears.

:::image type="content" source="media/navigate-list-view.png" alt-text="Screenshot in the Field Service web app showing a list of accounts.":::

| Number | Description |
|--------|-------------|
| 1      | **Go back**: Go back to the previous page. |
| 2      | **Select records**: Place a check mark in this column to select one or more records. Depending on the page you're on, you might be able to apply a single operation to all the selected records at once by using buttons in the command bar. |
| 3      | **Open a record**: Select a record to open its record view, which shows the record details. |
| 4      | **Sort or filter the list**: Select the arrow to sort or filter the list by values in that column. |
| 5      | **View selector**: Select to change views. |
| 6      | **Edit columns**: Add, remove, or reorder columns. Learn more in [Column editor](/power-apps/user/grid-filters-advanced#select-columns-in-the-column-editor) |
| 7      | **Edit filters**: See which filters are applied to the current view and add or modify them. Learn more in [Filter editor](/power-apps/user/grid-filters-advanced#select-filter-conditions-in-the-filter-editor) |
| 8      | **Filter by keyword**: Enter text in the search field to show only those records in the current view. |

## Work with record views

Record views show all available data and settings for a single record. They're also known as form views.

Open a record view by selecting a record that appears in a list view. You can also open a record view by following a link from a related record.

:::image type="content" source="media/navigate-record-view.png" alt-text="Screenshot in the Field Service web app showing an account record.":::

| Number | Description |
|--------|-------------|
| 1      | **Tabs**: Record views are divided into tabs. Each tab provides a collection of related fields from the record. The current tab is shown underlined. Select any tab name to go to that tab. |
| 2      | **Related**: Most records show a **Related** tab. This tab is a drop-down list of different types of records related to a displayed record. |
| 3      | **Heading bar**: Some record views display a few important fields in the heading bar, such as record owner. |
| 4      | **View and edit all field values**: In the main body of the record view, you find all of the fields related to the current tab. Fields marked with a red asterisk are required, and you can't save the record without valid values. Fields marked with a blue plus sign are especially important or recommended, but aren't strictly required. Fields showing a lock icon are read-only and can't be edited. |
