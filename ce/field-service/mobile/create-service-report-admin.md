# Enable technicians to create service reports

Use service reports in Dynamics 365 Field Service to document work completed, tasks performed, and products or parts used during service. Reports can be generated as customer-ready PDF files.

Administrators or developers can install and extend a reporting solution, add the required custom control and customizations, and enable service report creation for technicians.

## Install, extend, and enable the reporting solution

For detailed instructions, go to [Add a custom reporting control for service reports in Dynamics 365 Field Service](/dynamics365/guidance/resources/fs-reporting-solution-service-reports).
