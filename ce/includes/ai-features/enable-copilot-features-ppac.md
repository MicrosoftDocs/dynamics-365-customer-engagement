### Enable Copilot features

1. In the site map of Power Plaform admin center, select Copilot.
1. Select **Settings** > **Copilot** under **Dynamics 365 Customer Service**.
1. Search for the environment and select it, and then select **Edit setting**.
1. Select the **Enable AI-powered Copilot features** checkbox and save the changes.
