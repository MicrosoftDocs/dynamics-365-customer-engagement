### Enable AI agents

1. In the site map of Power Platform admin center, select Copilot.
1. Select **Settings** > **Agents** under **Dynamics 365 Customer Service**.
1. Search for the environment and select it, and then select **Edit setting**.
1. Select the **Enable AI agents** checkbox and save the changes.
