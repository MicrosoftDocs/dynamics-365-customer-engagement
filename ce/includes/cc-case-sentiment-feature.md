   - View case sentiment if your administrator enabled sentiment analysis for cases. Learn more in [Configure sentiment analysis for case](../customer-service/administer/configure-case-sentiment-analysis.md). The application doesn't display case sentiment in the following scenarios:
       - The case was created before the case sentiment feature was configured.
       - The case is created but no interaction is linked to it yet.
       - Sentiment isn’t generated due to an error.
