## Syntax

`Xrm.Encoding.htmlDecode(arg)`

## Parameters

|Parameter Name        | Type           | Required  |Description  |
| ------------- |-------------| -----|-----|
|arg        | String           | Required  |HTML-encoded string to be decoded.  |

**Type**: String

**Description**: Decoded string.
