# getDisabled (Client API reference)

[!INCLUDE[](../../../../includes/cc_applies_to_update_9_0_0.md)]

Returns whether the control is disabled.

## Syntax

`formContext.getControl(arg).getDisabled();`

## Return Value

**Type**: Boolean.

**Description**: true if disabled; false otherwise.
