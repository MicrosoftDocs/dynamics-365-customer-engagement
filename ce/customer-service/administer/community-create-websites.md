# Create new websites in Community
Community allows organizations to create multiple websites to support different audiences, scenarios, or branding needs.

## Create a new website

Complete the following steps to create a new website in Community:

1. Go to **Website Settings** in the site map.
2. Select **New** to create a website.
3. Go to [Power Pages](https://make.powerpages.microsoft.com/) to provision a website for the new Community site.
