# Use the prompt gallery in Service Agent

The prompt gallery helps you discover, save, and reuse prompts in Service Agent. You can browse Microsoft-provided prompts, save prompts that you use frequently, and share prompts with others in your organization.

## Prerequisites

Your administrator must configure Service Agent before the prompt gallery is available. Learn more in [Enable Service Agent](../administer/configure-service-agent.md).

## Open the prompt gallery

In the Service Agent pane, select **Prompt Gallery** at the bottom of the welcome screen. Or, select **See less** to expand the starter prompts view and find the gallery link.

The prompt gallery displays prompts organized by the following tabs:
   - **Suggested**: Prompts recommended by Microsoft for your role and common tasks.
   - **Your prompts**: Prompts you save.
   - **Team prompts**: Prompts shared by others in your organization.

## Use a prompt from the gallery

1. In the prompt gallery, browse or search for a prompt.
1. Select the prompt to add it to the Service Agent message box.
1. Edit the prompt if needed, and then press **Enter** or select **Send**.

## Save a prompt

You can save any prompt you use frequently so it's easy to find again.

1. After sending a message to Service Agent, select the **...** (more options) menu on the response or input.
1. Select the save option to add it to **Your prompts**.

The prompt is available from the **Your prompts** tab the next time you open the gallery.

## Share a prompt

1. Open a saved prompt from **Your prompts**.
1. Select the share option to make it available to others in your organization.

The prompt becomes available on the **Team prompts** tab for other users.

## Considerations

- Browse the **Suggested** tab to discover prompts for common customer service tasks.
- Save prompts that you use frequently to reduce repetitive typing.
- Share useful prompts with your team to help standardize common workflows.
- The prompts available in the gallery depend on how your administrator configured Service Agent.
