## Use Copilot in Dynamics 365 Sales and Sales agent in Microsoft 365 Copilot from within Dynamics 365 Sales

The Microsoft 365 Copilot chat experience includes both the Copilot in Dynamics 365 Sales skills and the Sales agent in Microsoft 365 Copilot. With this unified chat experience, you can seamlessly access all your chats and agents in Microsoft 365 across different apps. 

- To use the Copilot in Dynamics 365 Sales skills, select **Copilot** > **App Skills** in your Dynamics 365 Sales app. This option is only available in Dynamics 365 Sales. Learn more in [Ask Copilot questions](use-sales-copilot.md). 

- To use the Sales agent in Microsoft 365 Copilot, select **Copilot** > **Chat** and then select the Sales agent from the list of agents. This option is available across Microsoft 365 apps and in Dynamics 365 Sales. Learn more in [Use the Sales agent in Microsoft 365 Copilot from within Dynamics 365 Sales (preview)](use-sales-agent-in-m365-copilot.md).
