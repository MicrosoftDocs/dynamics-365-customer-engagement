# Use the Sales agent in Microsoft 365 Copilot from Dynamics 365 Sales

Get quick answers and actionable insights from your sales data by simply asking questions in natural language. Access the Sales agent in Microsoft 365 Copilot from within the Sales Hub app to find information faster, understand customer context, and make informed decisions - without leaving the Sales Hub app. Learn more about the scenarios supported, prompts you can use, and best practices in the [Sales agent in Microsoft 365 Copilot documentation](/microsoft-sales-copilot/sales-chat-overview).

## Access the agent from the Sales Hub app

1. In the Sales Hub app, select the **Copilot** icon in the top-right corner of the app.
1. From the dropdown, select **Chat** to open Microsoft 365 Copilot in the side pane.
1. Select the Hamburger menu in the top-left corner of the side pane to open the navigation pane.
1. Under **Agents**, select **Sales**. [Why don't I see the Sales agent?](faqs-sales-agent.md#why-dont-i-see-the-sales-agent)  
   :::image type="content" source="media/sales-agent-pane.png" alt-text="Screenshot of the Sales agent in Microsoft 365 Copilot in the Sales Hub app.":::
1. Start asking questions in natural language to get answers and insights from your sales data. [Why don't I get any responses from the Sales agent?](faqs-sales-agent.md#why-dont-i-get-any-responses-from-the-sales-agent)

You can access the Sales agent in Microsoft 365 Copilot from any Microsoft 365 application, such as Outlook, Teams, or Word, and any Dynamics 365 apps that support Microsoft 365 Copilot. For more information on using the agent, refer to the [Sales agent in Microsoft 365 Copilot documentation](/microsoft-sales-copilot/use-sales-chat?context=/dynamics365/context/sales-context).
