# Configure Power Platform Admin Center Copilot settings (preview)

[!INCLUDE [preview-banner](~/../shared-content/shared/preview-includes/preview-banner.md)]

The Power Platform Admin Center administrator can configure the Power Platform Admin Center Copilot settings for Dynamics 365 Sales from the Power Platform Admin Center. Admins can enable or disable the Power Platform Admin Center Copilot feature for a particular environment or environment groups.

[!INCLUDE [preview-banner](~/../shared-content/shared/preview-includes/preview-note-d365.md)]

## Prerequisites

You need to have the Tenant admin or Power Platform admin role to access the Power Platform Admin Center Copilot settings.

## Access Power Platform Admin Center Copilot settings

To access the Power Platform Admin Center Copilot settings:

1. Sign in to the [Power Platform Admin Center](https://admin.powerplatform.microsoft.com/).
2. In the left navigation pane, select **Copilot**.
3. Select **Settings**.
4. In the **Settings** pane, under **Dynamics 365 Sales**: 
    - Select **Copilot** to enable Copilot-powered features in Dynamics 365 Sales.
    - Select **AI Agents** to use AI Agents in Dynamics 365 Sales.
