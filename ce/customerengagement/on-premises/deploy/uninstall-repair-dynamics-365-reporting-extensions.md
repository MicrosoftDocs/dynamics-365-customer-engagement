# Uninstall or repair Dynamics 365 Reporting Extensions



To perform [!INCLUDE[pn_CRM_Reporting_Extensions_server_side](../includes/pn-crm-reporting-extensions-server-side.md)] maintenance, sign in to the computer where [!INCLUDE[pn_CRM_Reporting_Extensions_server_side](../includes/pn-crm-reporting-extensions-server-side.md)] is installed. Then, start **Programs and Features** in [!INCLUDE[pn_Control_Panel](../includes/pn-control-panel.md)] and select from the following options.  
  
-   **Uninstall**. To remove (uninstall) the [!INCLUDE[pn_CRM_Reporting_Extensions_server_side](../includes/pn-crm-reporting-extensions-server-side.md)], right-click **Microsoft Dynamics 365 Reporting Extensions**, select **Uninstall/Change**, select **Uninstall**, and then select **Uninstall**. Follow the instructions on the screen.  
  
-   **Repair**. To repair the [!INCLUDE[pn_CRM_Reporting_Extensions_server_side](../includes/pn-crm-reporting-extensions-server-side.md)], right-click **Microsoft Dynamics 365 Reporting Extensions**, select **Uninstall/Change**, select **Repair**, and then select **Next**. Follow the instructions on the screen.
