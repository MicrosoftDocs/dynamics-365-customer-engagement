# Manage servers



You can do the following actions in the **Servers** area of [!INCLUDE[pn_deploymentmanager](../includes/pn-deploymentmanager.md)]:  
  
-   View a list of [!INCLUDE[pn_microsoftcrm](../includes/pn-microsoftcrm.md)] servers.  
  
-   Enable or a disable a server.  
  
-   Delete a server.  
  
 You can customize the **Servers** area view to display the information that you want about the servers that are listed. Also, you can export a list of those servers and any displayed details to a text file.
