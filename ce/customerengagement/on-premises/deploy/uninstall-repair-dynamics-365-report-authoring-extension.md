# Uninstall or repair Dynamics 365 Report Authoring Extension



To perform Dynamics 365 Report Authoring Extension maintenance, select from the following options. Before you select an option, you must start **Programs and Features** in [!INCLUDE[pn_Control_Panel](../includes/pn-control-panel.md)].  
  
 Sign in to the computer where Dynamics 365 Report Authoring Extension is installed.  
  
-   **Uninstall**. To remove (uninstall) Dynamics 365 Report Authoring Extension, right-click **Microsoft Dynamics 365 Report Authoring Extension**, select **Uninstall/Change**, select **Uninstall**, and then select **Uninstall**. Follow the instructions on the screen.  
  
-   **Repair**. To repair Dynamics 365 Report Authoring Extension, right-click **[!INCLUDE[pn_crm_fetch_extension](../includes/pn-crm-fetch-extension.md)]**, select **Uninstall/Change**, select **Repair**, and then select **Next**. Follow the instructions on the screen.
